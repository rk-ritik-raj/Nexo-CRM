# Xeno Mini CRM

Xeno Mini CRM is a lightweight customer relationship management application designed for small teams and startups. It provides a clean interface for managing contacts, tracking sales opportunities, and organizing customer interactions.

## Key Features

- Contact management
- Lead and opportunity tracking
- Task and activity logging
- Simple sales pipeline visualization
- Search and filter for fast access
- Responsive layout for desktop and mobile

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Version Control: Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/xeno-mini-crm.git
   ```
2. Install dependencies:
   ```bash
   cd xeno-mini-crm
   npm install
   ```
3. Configure environment variables (if required).
4. Start the application:
   ```bash
   npm start
   ```

## Usage

- Open the app in your browser.
- Add and manage customer records.
- Create and track sales opportunities.
- Use the dashboard to review recent activity.

## Project Structure

- `client/` — frontend code and assets
- `server/` — backend API and server code
- `config/` — configuration files
- `README.md` — project documentation
